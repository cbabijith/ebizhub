# Admin App

EzhavaClub Admin Flutter Application

## Setup

1. Install Flutter: https://flutter.dev/docs/get-started/install
2. Install dependencies:
```bash
flutter pub get
```

3. Configure Supabase:
   - Update `lib/main.dart` with your Supabase URL and anon key

## Run

```bash
flutter run
```

## Build

```bash
# Android
flutter build apk

# iOS
flutter build ios
```
